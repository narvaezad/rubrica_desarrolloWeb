# rubrica_desarrolloWeb
# Uso responsable de redes sociales y ciberseguridad


## Descripción

Este proyecto consiste en el desarrollo de un sitio web informativo sobre el **uso responsable de las redes sociales y la ciberseguridad**. El objetivo es explicar los riesgos que pueden existir en internet, como el robo de información, el ciberacoso o la difusión de noticias falsas, y mostrar la importancia de utilizar las redes sociales de manera responsable.

## Objetivo

Promover el uso responsable de las redes sociales y generar conciencia sobre la importancia de proteger la información personal en internet.

## Estructura del sitio

El sitio web está dividido en varias secciones:

* Inicio
* Contexto
* Análisis
* Derechos y deberes ciudadanos
* Propuestas de solución
* Conclusión

## Tecnologías utilizadas

* HTML5
* CSS
